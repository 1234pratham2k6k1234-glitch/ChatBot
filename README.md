# 🤖 AI Chatbot using Groq + Streamlit

A real-time AI chatbot built using Groq API and Streamlit.

## 🚀 Features

* 💬 Chat interface
* ⚡ Fast responses using Groq LLMs
* 🧠 Context-based conversation
* 🔄 Model fallback system

## 🛠️ Tech Stack

* Python
* Streamlit
* Groq API (LLaMA / GPT OSS models)

## 🔑 Setup Instructions

1. Clone the repo:

```bash
git clone https://github.com/YOUR_USERNAME/chatbot-groq.git
cd chatbot-groq
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Set API key:

```bash
export GROQ_API_KEY="your_api_key"
```

(Windows PowerShell)

```powershell
$env:GROQ_API_KEY="your_api_key"
```

4. Run app:

```bash
streamlit run app.py
```

## 📌 Models Used

* llama-3.1-8b-instant
